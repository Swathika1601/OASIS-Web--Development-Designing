# Personal Portfolio Website

This project is a **Personal Portfolio Website** created using **HTML, CSS, and JavaScript**. It is designed as a beginner-friendly web development project that allows developers to showcase their skills, projects, and resume in a clean and organized way.

## 📌 Project Description

Creating a personal portfolio is one of the most common and important projects for beginners in web development. A portfolio website helps developers present their **skills, projects, and personal information** to potential employers or collaborators.

In this project, a portfolio webpage is built where:

- An **image and introduction** appear at the top of the page.
- An **About section** introduces the developer.
- A **Skills section** highlights technical abilities.
- A **Projects section** showcases sample work.
- A **Resume download button** allows users to access the developer's resume.
- A **Contact section** provides ways to get in touch.

This project helps beginners practice **HTML structure, CSS styling, layout design, and basic JavaScript interaction**.

## ▶️ How to Run the Project

1. Download or clone the project repository.
2. Open the project folder.
3. Open **index.html** in any web browser.

The portfolio website will load and display all sections including the introduction, skills, projects, and contact information.
