# Temperature Converter Website

A simple and interactive **Temperature Converter Web Application** built using **HTML, CSS, and JavaScript**. This project allows users to convert temperature values between **Celsius, Fahrenheit, and Kelvin** easily.

It is designed with a clean user interface and basic input validation to ensure users enter valid numeric values.

---

## 🚀 Features

* Enter temperature value using an input field
* Select temperature unit (Celsius, Fahrenheit, or Kelvin)
* Convert temperature instantly by clicking the **Convert** button
* Display converted results with correct units
* Input validation to ensure only numbers are accepted
* Clean and responsive user interface

---

## 🧩 User Interface Components

The website contains four main elements:

1. **Input Field**

   * Allows users to enter a temperature value.
   * Validates that the input is a numeric value.

2. **Dropdown Menu**

   * Lets users select the unit of the entered temperature:

     * Celsius (°C)
     * Fahrenheit (°F)
     * Kelvin (K)

3. **Convert Button**

   * Triggers the temperature conversion.

4. **Result Display Area**

   * Displays the converted temperature values along with their units.

---

## 🔁 Supported Conversions

The application supports the following conversions:

* Celsius → Fahrenheit
* Celsius → Kelvin
* Fahrenheit → Celsius
* Fahrenheit → Kelvin
* Kelvin → Celsius
* Kelvin → Fahrenheit

## ⚙️ How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open **index.html** in any web browser.
4. Enter a temperature value.
5. Select the temperature unit.
6. Click **Convert** to view the result.
